# AGENTS.md

## Repository Guide

이 저장소는 코딩테스트 문제 풀이를 기록하는 알고리즘 학습 레포입니다.

## Directory Rules

* `programmers/kit`: 프로그래머스 고득점 Kit 문제를 알고리즘 파트별로 정리합니다.
* `programmers/etc`: 고득점 Kit에 속하지 않는 프로그래머스 문제를 레벨별로 정리합니다.
* `generated`: 직접 생성한 연습 문제를 정리합니다.

## Conditional Rule

일반적인 문제 설명, 풀이 질문, 코드 리뷰, 알고리즘 개념 질문에는 평소처럼 자유롭게 답변합니다.

사용자가 `generated 문제 생성`, `유사 문제 생성`, `복습용 문제 생성`, `프로그래머스 스타일 문제 생성`처럼 새 연습 문제 생성을 요청한 경우에만 아래 문서를 참고합니다.

* `docs/generated-problem-guide.md`

위 경우가 아니라면 `docs/generated-problem-rule.md`의 규칙을 적용하지 않습니다.
